Визуализатор алгоритма Косарайю для поиска компонент сильной связности в графе.

Данная программа предоставляет возможность задания произвольного графа с последующей визуализацией работы алгоритма на нём, а также выводом сопутствующей пояснительной информации.

Для задания графа предоставлены следующие возможности

  • Ввод из файла:

На строке меню нажмите File -> Open, а затем выберите нужный файл. Для корректной работы программы граф в файле должен быть задан следующим образом: на первой строке расположено кол-во вершин в графе, далее построчно ребра вида i j, где i и j в диапазоне от 0 до кол-ва вершин.

  • Добавление вершин и задание ребер с помощью графического интерфейса

Добавить вершину в граф можно нажатием на кнопку Add vertex. Для добавления ориентированного ребра между вершинами нужно нажать на начальную вершину и перетянуть курсор к конечной вершине ребра.

С помощью графического интерфейса можно изменять созданный граф.

  • Удаление вершины/ребра из графа. Для этого требуется выбрать соответствующую вершину/ребро и нажать на кнопку Delete vertex.
  • Полное удаление графа. Для этого требуется нажать на кнопку Clear.
  • Изменение начальной/конечной вершины ребра. Для этого требуется выбрать ребро, а затем перетянуть его начало/конец к нужной вершине.
  • Для расположения вершин в правильном порядке нужно нажать File -> Sort graph.

Управление визуализацией работы алгоритма:

  • Для запуска работы алгоритма нужно нажать кнопку Start.
  • Чтобы временно остановить работу алгоритма требуется нажать на кнопку Pause.
  • Для возобновления работы алгоритма, после нажатия кнопки Pause, нужно повторно нажать на кнопку Start.
  • Для полной остановки работы алгоритма требуется нажать на кнопку Stop. В этом случае работа алгоритма прерывается, а граф возвращается в исходное состояние.
  • Также имеется возможность изменять скорость работы алгоритма. Для увеличения/уменьшения скорости нужно нажать соответственно +/-.

Управление окном приложения:

  • При изменении размера окна и холста меняется расстояние между вершинами.
  • Для закрытия приложения нужно нажать File -> Exit. Граф не сохраняется.

В процессе выполнения алгоритма в текстовом поле выводится сопутствующая информация о начале каждого его этапа. Для каждого этапа дополнительно выводится соответствующая информация. После завершения работы алгоритма все вершины, входящие в одну компоненту сильной связности, окрашиваются в определенный цвет (для каждой компоненты сильной связности он различен). Информация о количестве найденных компонент и о вершинах, входящих в эти компоненты, также будет выведена в текстовом поле.
